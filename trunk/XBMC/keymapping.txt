XBMC key mappings:
	 general key mappings:
		  page up/down
				  controller: left/right trigger
				  remote		: REVERSE/FORWARD
				  
      up/down/left/right
					controller: use DPAD
					remote    : use up/up/left/right
					
			selecting stuff:
					controller: A button
					remote    : select 
					
					
   my pictures:
      when viewing a picture/slideshow
        left/right/up/down : move when in zoom mode
        left/right         : next/prev picture when not in zoom mode
        black              : show file info
        START              : pause/continue
        X                  : zoom out
        A                  : zoom in 
   
   my files:
      switch between source/destination window
					controller:BLACK			
					remote    :DISPLAY
      
      (de)select files/directory
					controller: Y button
					remote    : play button
   
   my music:
			switch between files & playlist
					controller:L/R trigger/BLACK			
					remote    :DISPLAY

			add/remove file 2/from playlist
					controller: Y button
					remote    : play button
					
			find album info for file	
					controller: WHITE button
					remote    : info button	
					
   my videos:
			find IMDB info for file
					controller: WHITE button
					remote    : info button
